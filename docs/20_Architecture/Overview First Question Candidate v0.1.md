# Overview First Question Candidate v0.1

## Purpose

This is not an Overview layout draft.

This is not a homepage plan.

It only records the current product question:

**what is the first thing the runner truly wants to know when opening Overview?**

## Risk

Overview is more dangerous than Weekly or Monthly.

It is the page most likely to collapse into:

- a KPI summary
- a mini Garmin
- a little of everything

That would create information density without product meaning.

Overview should not answer:

`what data do we have?`

It should answer:

`what deserves attention now?`

## Candidate Questions

The following candidates are intentionally broader than one wording preference.

They represent different product worldviews.

### Q1

`我現在整體狀況如何？`

Strength:

- intuitive
- familiar
- easy to design around

Risk:

- drifts toward health dashboard language
- sounds closer to Garmin than to a coach

### Q2

`我現在該注意什麼？`

Strength:

- focused
- naturally leads to explanation

Risk:

- can become warning-oriented
- may overemphasize risk

### Q3

`我現在最值得做的一件事是什麼？`

Strength:

- coach-like
- actionable

Risk:

- may skip diagnosis and jump too quickly to action

### Q4

`今天，我最該關心的是什麼？`

Strength:

- focuses attention instead of reporting status
- narrows the page to one live priority
- naturally creates a second question: `為什麼？`

Risk:

- requires the page to be selective
- does not allow neutral information stacking

### Q5

`今天，最不該分心的是什麼？`

Strength:

- strong editorial focus
- naturally reduces clutter

Risk:

- slightly indirect
- may feel too designed if not grounded in real usage

### Q6

`如果教練現在只提醒一句，他會提醒什麼？`

Strength:

- highly distinctive
- strong coaching tone

Risk:

- sounds like a device for presentation more than a runner's real question

### Q7

`我現在最需要守住的是什麼？`

Strength:

- strong for recovery and taper contexts
- coach-like

Risk:

- less natural during stable or expansion periods

### Q8

`我現在最值得放心的是什麼？`

Strength:

- emotionally supportive

Risk:

- too narrow as a universal homepage frame

## First Elimination

The following candidates are weaker as the primary opening question:

- Q6: too meta; sounds more like presentation than inner user need
- Q8: emotionally useful, but too narrow for all states
- Q7: good in some contexts, but not broad enough as the default frame

## Leading Direction

The strongest current direction is:

`今天，我最該關心的是什麼？`

Why it stands out:

1. it prevents Overview from becoming a data collage
2. it defines Overview as an attention manager, not a status board
3. it creates a natural next question: `為什麼？`
4. it complements the other pages without overlapping them

## Relationship to Other Pages

If Monthly answers:

`我現在在哪？`

and Weekly answers:

`這週，我到底練到了什麼？`

then Overview may answer:

`今天，我最該關心的是什麼？`

This would make the four major pages functionally distinct:

| Page | Core question | Cognitive function |
| --- | --- | --- |
| Overview | 今天，我最該關心的是什麼？ | Attention |
| Weekly | 這週，我到底練到了什麼？ | Learning |
| Monthly | 我現在在哪？ | Position |
| Journey | 我正在成為什麼樣的跑者？ | Identity |

## What Q4 Would Change

If Q4 leads, Overview stops behaving like a dashboard homepage.

It becomes an attention surface.

That means the page would no longer try to show everything important.

It would try to identify:

- the single current coaching focus
- the reason that focus matters
- the one nearby surface worth opening next

Examples:

- `今天最該關心的是恢復。`
- `今天最該關心的是把節奏守住。`
- `今天最該關心的是長跑鞋的更換。`

The point is not whether these examples are final.

The point is that each one immediately creates the second question:

`為什麼？`

## Current Position

Q4 is the leading candidate.

Not because it is already correct.

But because it best protects Overview from becoming an information page instead of a product with a role.

## Next Step

Do not design Overview yet.

Reduce the candidate field further, then test whether Q4 can stand on its own before any layout grows around it.
