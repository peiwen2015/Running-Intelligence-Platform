# Weekly First Question Candidate v0.1

## Purpose

This is not a Weekly design guideline.

It does not try to define Weekly up front.

It only records the current product finding:

**the leading candidate for Weekly's first question.**

## Context

Monthly succeeded after it found its first real question:

`我現在在哪？`

Weekly should not inherit Monthly's answer.

It should discover its own.

The question is not:

- what cards should Weekly contain?
- what charts should Weekly show?
- how should Weekly look?

The question is:

**When the runner opens Weekly, what is the first thing they truly want to know?**

## Candidate Review

Two candidates survived the first reduction:

### Q1

`這週最大的收穫是什麼？`

This leads naturally to a coach summary.

It is warm, direct, and readable.

But it tends to frame Weekly around a result.

### Q2

`這週，我到底練到了什麼？`

This leads naturally to adaptation and learning.

It works for:

- build weeks
- recovery weeks
- stable weeks
- imperfect weeks

It does not require the week to feel impressive.

It only requires the week to have taught the body something.

## Current Decision

Q2 is the leading candidate.

Not because it sounds better.

But because it better matches the worldview already established across:

- Recovery Domain
- Load Build Domain
- Narrative Engine

All three are centered on:

`Training → Adaptation → Learning`

Weekly should likely live in the same frame.

## What Q2 Changes

If Weekly starts from:

`這週，我到底練到了什麼？`

then the page is no longer centered on:

- completion
- mileage
- single-session achievement

It becomes centered on:

- adaptation
- what the body learned
- what rhythm was preserved
- what ability quietly grew

This means Weekly becomes different from Monthly.

Monthly answers:

`我現在在哪？`

Weekly would answer:

`這七天，身體真正學會了什麼？`

## Natural Reading Sequence

If Q2 holds, Weekly may grow in this order:

```text
這週，我到底練到了什麼？
   ↓
哪一堂課真正教會了我？
   ↓
哪些證據支持這個學習？
   ↓
下週怎麼把這個能力繼續長大？
```

This is not yet a layout.

It is only the first candidate reading sequence implied by Q2.

## Relationship to Q1

Q1 should not be discarded.

It may still belong in Weekly, but not as the opening question.

Possible role:

- Q2 defines the page's core question
- Q1 becomes the coach's summary of that learning

In that sense:

`這週最大的收穫是什麼？`

may survive as a second-layer coaching statement.

## Example Week

Given a week like:

- Wednesday tempo session
- HRV dropped
- weekend LSD was cancelled
- replaced with 70-minute easy run
- overall rhythm recovered by the end of the week

Q2 produces a more adaptation-centered reading:

> 這週，你練到的不是耐力突破。  
> 你練到的是：在狀態不完美時，仍然把整體節奏留住。

This is stronger than simply saying:

> 這週最大的收穫是節奏穩住了。

Because it explains not only the result, but the capability that was built.

## Validation

Q2 should keep leading only if it can naturally support:

1. recovery weeks
2. build weeks
3. disrupted weeks
4. one clear next step

If it fails to hold across those week types, it has not yet earned promotion.

## Next Step

Do not design Weekly yet.

Use Q2 to grow one minimal Weekly page skeleton.

Then test whether the rest of the page emerges naturally from it.
