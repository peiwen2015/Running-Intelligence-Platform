# Weekly Q2 Validation v0.1

## Purpose

This is not a Weekly layout draft.

This is not a Weekly writing draft.

This is only a validation slice for one question:

`這週，我到底練到了什麼？`

The goal is to test whether Q2 can stand on its own before any page structure is designed around it.

## Validation Rule

Each case is allowed to answer Q2 in two sentences only:

1. not what was completed
2. what the body or the runner actually learned

Nothing else is added:

- no why
- no evidence
- no next step
- no UI

If the reader naturally nods before asking "為什麼？", then Q2 remains alive.

## Case 1 | Build Week

**這週，你練到的不是跑得更多。**

**你練到的是：開始承受更大的訓練刺激。**

## Case 2 | Recovery Week

**這週，你練到的不是增加訓練。**

**你練到的是：讓前面的刺激真正留下來。**

## Case 3 | Disrupted Week

**這週，你練到的不是完成原本的計畫。**

**你練到的是：在狀態改變時，仍然守住整體節奏。**

## Reading Test

Q2 stays in the lead only if all three cases feel true without explanation.

The test is simple:

- does the sentence feel natural?
- does the reader nod before asking why?
- does it still sound like coaching?
- does it still hold when the week was imperfect?
- does it make the next question appear by itself?

If yes, then Weekly has not yet found its page, but it may have found its first question.
