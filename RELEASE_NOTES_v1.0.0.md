# CoachOS v1.0.0

CoachOS 的第一個正式版本，正式把資料匯入、產品語意和對外入口收斂成一條穩定的本機產品鏈路。

## Highlights

- `資料匯入工具` 納入產品流程，負責 `FIT -> Excel -> SQLite`
- `CoachOS` 成為正式對外主名稱
- `Semantic Layer v1.0` 與 `SQLite Schema v1.0` 完成主要產品鏈路
- `Metadata Repository v1.1` 成為名詞與規則的治理基礎
- `AI 延伸分析` 支援把整理結果複製給 AI，並可將回覆寫回頁面

## Product Surfaces

- `Activity`
- `Overview`
- `Weekly`
- `Monthly`

`Journey` 目前保留在程式裡，但暫時不作為公開產品面。
